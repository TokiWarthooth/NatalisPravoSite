# Директория для изображений

## Инструкция по добавлению фотографий юриста

Сохраните фотографии в эту директорию со следующими именами:

1. **lawyer-main.jpg** - Фото для главной страницы (круглое)
   - Рекомендуемый размер: 280x280px или больше (квадратное)
   - Используется первое фото (с книгой)
   - Будет обрезано в круг автоматически

2. **lawyer-about.jpg** - Фото для страницы "О юристе"
   - Рекомендуемый размер: 250x300px или больше (портретное)
   - Используется второе фото (сидя в кресле)
   - Будет обрезано с закругленными углами

## Формат файлов
- Поддерживаются форматы: JPG, JPEG, PNG
- Рекомендуемый формат: JPG для лучшей производительности
- Качество: высокое (не сжимайте слишком сильно)

## Как добавить фотографии

### Вариант 1: Через файловый менеджер
1. Откройте директорию `public/images/`
2. Скопируйте туда фотографии с правильными именами

### Вариант 2: Через командную строку
```bash
# Из корня проекта
cp /путь/к/фото1.jpg public/images/lawyer-main.jpg
cp /путь/к/фото2.jpg public/images/lawyer-about.jpg
```

### Вариант 3: Через Docker (если файлы на хосте)
```bash
docker cp /путь/к/фото1.jpg natalispravo-nginx-1:/var/www/html/public/images/lawyer-main.jpg
docker cp /путь/к/фото2.jpg natalispravo-nginx-1:/var/www/html/public/images/lawyer-about.jpg
```

После добавления фотографий обновите страницу в браузере (Ctrl+F5 для полной перезагрузки).
